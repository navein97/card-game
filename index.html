Card Game with 4 Buddies

Condition:
- Player with the highest number of cards with same alphanumeric part (i.e., K@, K#, K^, K*).
- If more than 1 player has the same number of winning cards, the alphanumeric part with higher value won. If tie, the symbol part with high value won.
(K@ > K# > K^ > K*)

*Note: It didn't state that A is the strongest.
